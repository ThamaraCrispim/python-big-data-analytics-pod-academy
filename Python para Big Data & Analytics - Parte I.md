## O que é Python?

Python é uma linguagem de programação de alto nível, interpretada, orientada a objetos e de propósito geral.  
Criada em 1991 por Guido van Rossum, se destaca pela sintaxe clara e simples, sendo uma das linguagens mais populares da atualidade.

É amplamente utilizada devido à sua facilidade de aprendizado, grande comunidade ativa e vasta biblioteca de módulos e frameworks, que facilitam o desenvolvimento em diversas áreas.

## Python puro x Anaconda

**Python puro**  
Refere-se à instalação padrão baixada do site oficial do Python. Inclui a linguagem e a biblioteca padrão, com módulos essenciais para programação geral.

**Anaconda**  
Distribuição do Python voltada para ciência de dados, incluindo pacotes, ferramentas e o gerenciador `conda`. Facilita instalar bibliotecas como NumPy, Pandas, Matplotlib e frameworks de machine learning.

### Como instalar
- **Python puro:** disponível em *python.org*  
- **Anaconda:** disponível em *anaconda.com*

## IDEs (Ambientes de Desenvolvimento)

IDE é um software que oferece ferramentas que facilitam o desenvolvimento, como:
- editor com auto-completar  
- depuração  
- gerenciamento de projetos  
- integração com Git, ambientes virtuais e bibliotecas  

### IDEs populares

#### PyCharm
IDE da JetBrains, muito usada para Python. Possui:
- versão gratuita (Community)  
- suporte a Django, Flask, Pyramid  
- integração com Git  
- ferramentas de teste e depuração  

#### Spyder
IDE de código aberto voltado para computação científica.  
Características:
- editor com preenchimento automático  
- console interativo  
- depurador  
- suporte a Matplotlib e Seaborn

## Editores de Código

Editores são mais leves que IDEs, usados apenas para edição e organização de código.

### Exemplos

#### Visual Studio Code
- Editor leve, extensível e multiplataforma  
- Suporte excelente para Python

#### Sublime Text
- Editor rápido e minimalista  
- Suporte a diversas linguagens

## Boas práticas – PEP8

PEP8 é o guia oficial de estilo do Python, criado em 2001.  
Inclui recomendações sobre:
- indentação  
- nomes de variáveis e funções  
- espaçamentos  
- tamanho máximo de linha  
- comentários  
- organização do código  

Seguir o PEP8 ajuda a manter o código limpo, legível e fácil de manter.
